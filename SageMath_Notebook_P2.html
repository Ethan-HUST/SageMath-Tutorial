Ứng dụng trong Giải tích 1¶

Các câu lệnh cơ bản thường dùng¶

  • var('x'): Khai báo biến.
  • solve(...): Giải phương trình hoặc bất phương trình.
  • limit(...): Tính giới hạn.
  • diff(...): Tính đạo hàm.
  • integrate(...): Tính tích phân.
  • plot(...), plot3d(...): Vẽ đồ thị 2D, 3D.

1. Phép tính vi phân hàm một biến số¶

Tìm tập xác định của hàm số¶

Hàm: $y = \sqrt{2\ln{x} - 3}$

Out[2]:

Phương trình vi phân tuyến tính¶

Giải: $ y' + y = e^x $

In [38]:
var('x y')
y = function('y')(x)
de = diff(y, x) + y == e^x
desolve(de, y)
Out[38]:
1/2*(2*_C + e^(2*x))*e^(-x)

Giải: $ y'' - y = 0 $

In [39]:
y = function('y')(x)
de2 = diff(y, x, 2) - y == 0
desolve(de2, y)
Out[39]:
_K2*e^(-x) + _K1*e^x

Phương trình Euler¶

Giải: $ x^2 y'' - x y' + y = 0 $

In [41]:
y = function('y')(x)
de = x^2*diff(y, x, 2) - x*diff(y, x) + y == 0
desolve(de, y)
Out[41]:
(_K2*log(x) + _K1)*x

Hệ phương trình vi phân cấp 1¶

Giải: $ \begin{cases} x' = x + y \\ y' = x - y \end{cases} $

In [42]:
t = var('t')
x = function('x')(t)
y = function('y')(t)
de1 = diff(x, t) == x + y
de2 = diff(y, t) == x - y
desolve_system([de1, de2], [x, y])
Out[42]:
[x(t) == 1/2*sqrt(2)*(x(0) + y(0))*sinh(sqrt(2)*t) + cosh(sqrt(2)*t)*x(0),
 y(t) == 1/2*sqrt(2)*(x(0) - y(0))*sinh(sqrt(2)*t) + cosh(sqrt(2)*t)*y(0)]

Biến đổi Laplace¶

Phân tích hồi quy tuyến tính¶

Giải hệ phương trình để tìm hệ số hồi quy.

In [87]:
var('x')
data = [(1, 2), (2, 4), (3, 6)]
m, b = var('m b')
model = m * x + b
eqs = [model.subs(x=xv) == yv for xv, yv in data]
solve(eqs, m, b)
Out[87]:
[[m == 2, b == 0]]
